# Media Sorter

A simple web application for sorting media into approved and declined folders.

## How It Works

1. Place media in the `toSort` folder
2. Open the web interface in your browser
3. Review each media file and click "Approve" or "Decline"
4. Approved media move to the `approved` folder
5. Declined media move to the `declined` folder

## Supported Formats

Currently it only supports images with the following formats: JPG, JPEG, PNG, WebP.

## About This Project

This application was developed using AI with human programming interventions to explore AI capabilities and limitations in software development.

## Getting Started

```bash
npm install
npm start
```

Then open `http://localhost:3000` in your browser.
